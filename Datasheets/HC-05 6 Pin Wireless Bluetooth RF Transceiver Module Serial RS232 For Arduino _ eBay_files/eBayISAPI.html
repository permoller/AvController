HC-05 6 Pin Wireless Bluetooth RF Transceiver Module Serial RS232 For Arduino


Material: PC
Color: with the chassis HC-05 master and slave integrated
First, the module parameters:
1.PCB size: 37.3mm (length) * 15.5mm (width)
2. Weight: 3.5g
3. Input voltage: 3.6V - 6V, prohibit more than 7V
4. Power supply anti-reverse, then the module does not work
5. 6 feet: EN / VCC / GND / RXD / TXD / STATE (Bluetooth status pin, not connected to the output low, connected to the output high)
6. With the connection status indicator, LED flash means no Bluetooth connection; LED slow flash to enter the AT command mode
7. On-board 3.3V regulator chip, input voltage DC 3.6V-6V; unpaired, the current is about 30mA (due to LED flashing, the current is in a state of change); matching success, the current is about 10mA
8. Interface level 3.3V, you can directly connect a variety of single-chip (./51/AVR/PIC/ARM/MSP430, etc.), 5V microcontroller can also be directly connected. Directly connected to the microcontroller serial port, not through the MAX232 chip!
9. Effective transmission distance of 10 meters, more than 10 meters is also possible, but do not guarantee the quality of the connection
10. After successful pairing, can be used as a duplex serial port. No need to understand any Bluetooth protocol, but only supports 8-bit data bits, 1 stop bit, no parity communication format. This is also commonly used in Z communication format, do not support other formats!
11. Press the button to power the Bluetooth module can enter the AT mode, set the parameters and query information
12. The host and slave modes can be switched via the AT command
13. The default baud rate is 9600 and the default pairing password is 1234. The default name is HC-05
14. With transparent heat shrink tubing protection
Second, the wiring method:
VCC: Connect the power supply positive
GND: Connect the power supply negative
RXD: the receiving end, the Bluetooth module receives the data sent from other devices; normally send the TXD of the other device
TXD: the sender, the Bluetooth module to send data to other devices; normal access to other devices receiving RXD
EN: Enabled, need to enter AT mode when connected to 3.3V
Weight: 4g
Quantity: 1 Bluetooth module
